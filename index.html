Estrutura Básica do HTML

Para escrevermos um codigo em HTML é necessário digitar a tag <html> e fecha-la

repetindo o nome da tag porem com um sinal de "/" antes da palavra "HTML" como por exemplo:

</html>

Dentro do escopo da tag, é necessário escrever as tags <head> e <body> deve ser feito isso pois

o head ira armazenar informações como o titulo da pagina ou o icone, e o body irá apresentar o cont-

eudo de nossa pagina. Lembrando que o titulo da pagina pode ser alterado utilizando a tag <title>

Falando sobre tags

As tags tem como finalidade estruturar o nosso conteudo em um documento HTML

desde apenas apresentar o conteudo do lado via cliente, para deixa-lo mais at-

raente ao usuario ate deixa-lo mais semantico, algo que e muito importante para

mecanismos de busca, atraves da otimização SEO.

Algumas das tags mais utilizadas no HTML:

  1. <h1>: tem como finalidade criar um titulo na pagina
  2. <p>: tem como finalidade criar um paragrafo na pagina
  3. <a>: tem como finalidade criar um link na pagina
  4. <script>: tem como finalidade executar um script de programacao na pagina, ou chamar um documento de script utilizando o src= na pagina

Lembrando que devido algumas mudancas entre as versoes do HTML, algumas tags podem aca-

bar perdendo o suporte, como por exemplo a tag <font> no qual foi muito uti-

lizada no HTML4 porem atualmente no HTML5 usamos a tag <style> para dar esti-

los em textos.

Atributos das Tags

Os atributos das tags sao aditivos que podemos colocar nas tags com um respectivo objetivo

Por exemplo, temos os atributos "class" e "id" dos quais funcionam como identificadores uni-

cos para determinado elemento no HTML.

Algumas tags possuem atributos unicos para seu tipo, por exemplo a tag <input> recebe o atributo "type="

que vem com objetivo de informar para o navegador qual o tipo de input e dados se espera na entrada.

Textos

Os textos sao um elemento muito importante na estrutura HTML pois cerca de 70% a 80%

de um site e composto por eles. No HTML5 possuimos algumas tags para estruturar esse

tipo de conteudo em nosso site, como por exemplo

  • <p>: Tem como objetivo criar um paragrafo de texto.
  • <br>: Tem como objetivo quebrar a linha no final do paragrafo.
  • <h1 - h6>: Tem como objetivo gerar um titulo na pagina, indo do maior(h1) ao menor(h6)

Listas Ordenadas e Nao Ordenadas

As listas sao estruturas fundamentais em nosso site, comumente usadas como menus de navegação ou tam-

bem em sites de receitas ou passo a passo em tutoriais.

No HTML possuimos dois tipos de listas, as Ordenadas, criadas com a tag <OL> e as Nao Ordenadas

criadas com a tag <UL>.

Dentro de ambas as tags e necessario utilizar a tag <li> para adicionarmos conteudos dentro de

nossa lista

Exemplos de listas com OL:

    Como ligar o carro

  1. Abrir a porta.
  2. Checar retrovisores.
  3. Colocar o cinto.
  4. Colocar a chave no local de entrada.
  5. Virar a chave.

Exemplo de listas com UL:

    Items escolares

  • Mochila
  • Caderno
  • Lapis
  • Caneta
  • Borracha

Links

Os link no HTML servem para levar o usuario a algum site ou ponto da pagina especificos

A tag utilizada para utilizar os link seria a tag <a> atraves dela utiliz-

amos o atributo "href=" e passamos o link da pagina no qual queremos que o usuario acesse

Exemplo:

FORMADA ERRADA: Acesse: https://www.dio.me/

FORMADA CORRETA: acesse aqui o site da DIO
